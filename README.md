# DTPageControl

This is a PageControl component for macOS. Updated from [NSPageControl](https://github.com/nerd0geek1/NSPageControl).

![](https://i.imgur.com/q8ZArtm.png)

## Installation
Install using Swift Package Manager.

## Requirements
- OS X 10.14+
- Xcode 11.0 or above
